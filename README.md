## 🎮 游戏简介

**《异形工厂》**是一款**模拟工厂建设**的策略游戏，玩家需要利用各种**工业工具**来规划和搭建自己的工厂，以高效开采矿石、加工物品，并完成挑战任务，最终赢取**终极奖杯🏆**！  

## 🏗️ 游戏玩法  

1. **拖拽工具**在地图上放置工厂设备，合理布局提高效率。
2. **采矿、加工、运输**，不同的工具各司其职。
3. **完成关卡任务**，升级你的工厂，让运作更高效！
4. **策略升级**，解锁新的能力，迎接更高难度挑战！

## 🏭 主要游戏元素  

### 🔧 工业工具  

- **开采器** ⛏️：放置在矿石上可**开采对应矿石**。
- **传送带** 🚛：运输矿石至交付中心，合理规划路径。
- **切割机** ✂️：可**切割圆形矿石**，分为两路运输。
- **垃圾桶** 🗑️：用于**回收不需要的矿石**，防止资源浪费。
- **戴夫的铁铲** 🔩：清除已放置的设备，便于调整布局。

### 🎮 操作方式  

- **W/A/S/D**：在拖拽工具时**调整方向**。
- **R + 方向键**：用于**旋转传送带**，设定不同的转向。
- **鼠标拖拽**：选择并放置工厂设备。

### 🔄 升级与任务  

- **开采器、传送带、切割机可升级**，提高运作效率。
- **商店升级**：使用金币提升**交付中心**价值或清除障碍。
- **任务目标**：每个关卡需要**开采一定数量的特定矿石**，通过三关即可获胜！

## 💾 如何运行  

1. 下载游戏文件。  
2. 运行 `Special_shapedFactory.exe` 文件。  
3. 拖拽工具，建设工厂，开始挑战！
